# Exercaise

Веб-приложение для проведения тренировок с темповым контролем. Работает полностью в браузере, не требует серверной части.

**https://exercaise.pavelkorolev.xyz/**

## Как устроена тренировка

Тренировка — это упорядоченный список упражнений. Каждое упражнение описывается набором параметров:

| Параметр | Описание |
|----------|----------|
| `title` | Название упражнения |
| `description` | Текстовое описание (опционально) |
| `repeat_count` | Количество повторений в подходе |
| `set_count` | Количество подходов |
| `rest_seconds` | Отдых между подходами (секунды) |
| `tempo` | Темп выполнения: `to`, `hold`, `from` (секунды на каждую фазу) |

### Темп

Каждое повторение разбито на три фазы:

- **to** — движение в одну сторону (например, опускание в присед)
- **hold** — задержка в конечной точке
- **from** — движение обратно

Плеер отсчитывает каждую фазу и автоматически переключает на следующую.

### Логика плеера

Плеер проходит через состояния в следующем порядке:

```
описание
  → повторение (to → hold → from) × N
  → отдых
  → следующий подход ...
→ следующее упражнение ...
→ завершение
```

- Если у упражнения есть `description` — перед началом показывается экран чтения с автоматическим таймером
- После всех повторений в подходе запускается таймер отдыха
- После последнего подхода — переход к следующему упражнению
- После последнего упражнения — экран завершения

### Формат JSON

Тренировки можно создавать вручную или генерировать с помощью AI (ChatGPT, Claude и др.) — достаточно описать желаемую тренировку и попросить сформировать JSON по схеме ниже. Готовый JSON импортируется в приложение.

```json
{
  "title": "Утренняя зарядка",
  "exercises": [
    {
      "title": "Приседания",
      "description": "Держи корпус ровно и не опускай таз ниже колен",
      "repeat_count": 10,
      "set_count": 3,
      "rest_seconds": 30,
      "tempo": { "to": 2, "hold": 1, "from": 2 }
    }
  ]
}
```

## Возможности

- **Звуковое сопровождение** — синтезированные звуковые сигналы (Web Audio API) для переходов между фазами, подходами и упражнениями.
- **Полноэкранный режим** — увеличенный интерфейс для просмотра на расстоянии.
- **Wake Lock** — экран не гаснет во время тренировки.
- **Клавиатурное управление** — Space (пауза), стрелки (навигация по подходам), Shift+стрелки (навигация по упражнениям), F (полноэкран), M (звук), Esc (стоп).
- **Импорт / редактирование** — импорт тренировок из JSON, редактирование через JSON-редактор.
- **Предустановленные тренировки** — несколько готовых тренировок для быстрого старта.
- **Оценка длительности** — автоматический расчёт примерной длительности тренировки.
- **Адаптивная вёрстка** — поддержка мобильных устройств, планшетов, десктопа и ландшафтной ориентации.

## Стек

- React 19, TypeScript, Vite
- Zustand (стейт), React Router 7 (маршрутизация)
- Web Audio API (звук), CSS Modules (стили)
- Vitest + Testing Library (тесты)

## Структура проекта

```
src/
  features/workout/
    model/       — типы, валидация, маппинг, расчёт длительности, хук плеера
    store/       — Zustand-стор
    components/  — WorkoutList, ExerciseList, WorkoutPlayer
  shared/
    ui/          — Button, Card, IconButton, Modal, MarqueeText
    storage/     — обёртка над localStorage
    sound/       — синтез звуков через AudioContext
    hooks/       — useWakeLock, useScrollIndicators
    i18n/        — плюрализация
```
